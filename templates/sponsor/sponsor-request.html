Request for GitHub sponsoring

GitHub can sponsor one Django Girls workshop per city with a 250$ grant. Applications must be received 2 months before the date of your workshop to be considered.

Before applying, your website needs to be ready and online. Then, you need to check if an event already happened in your city and if they received money from GitHub (check their sponsors section).

If your event is the first in your city or if the previous ones didn't receive sponsoring from GitHub, you can apply.

Note: Filling this form will be long. You'll have to download and edit some documents (invoice, bank information, etc).

We send those requests once every two weeks to GitHub. Next deadline: {{ deadline }}.
