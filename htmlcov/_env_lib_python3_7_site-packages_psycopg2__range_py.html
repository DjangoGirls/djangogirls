215 schema-qualified 

216 :param pyrange: a `Range` strict subclass, or just a name to give to a new 

217 class 

218 :param conn_or_curs: a connection or cursor used to find the oid of the 

219 range and its subtype; the typecaster is registered in a scope limited 

220 to this object, unless *globally* is set to `!True` 

221 :param globally: if `!False` (default) register the typecaster only on 

222 *conn_or_curs*, otherwise register it globally 

223 :return: `RangeCaster` instance responsible for the conversion 

224 

225 If a string is passed to *pyrange*, a new `Range` subclass is created 

226 with such name and will be available as the `~RangeCaster.range` attribute 

227 of the returned `RangeCaster` object. 

228 

229 The function queries the database on *conn_or_curs* to inspect the 

230 *pgrange* type and raises `~psycopg2.ProgrammingError` if the type is not 

231 found. If querying the database is not advisable, use directly the 

232 `RangeCaster` class and register the adapter and typecasters using the 

233 provided functions. 

234 

235 """ 

236 caster = RangeCaster._from_db(pgrange, pyrange, conn_or_curs)

480# Special adaptation for NumericRange. Allows to pass number range regardless 

481# of whether they are ints, floats and what size of ints are, which are 

482# pointless in Python world. On the way back, no numeric range is casted to 

483# NumericRange, but only to their subclasses 

484 

485class NumberRangeAdapter(RangeAdapter): 

486 """Adapt a range if the subtype doesn't need quotes.""" 

487 def getquoted(self): 

488 r = self.adapted 

489 if r.isempty: 

490 return b"'empty'" 

491 

492 if not r.lower_inf: 

493 # not exactly: we are relying that none of these object is really 

494 # quoted (they are numbers). Also, I'm lazy and not preparing the 

495 # adapter because I assume encoding doesn't matter for these 

496 # objects. 

497 lower = adapt(r.lower).getquoted().decode('ascii') 

498 else: 

499 lower = '' 

500 

501 if not r.upper_inf: 

502 upper = adapt(r.upper).getquoted().decode('ascii') 

503 else: 

504 upper = '' 

505 

506 return ("'%s%s,%s%s'" % ( 

507 r._bounds[0], lower, upper, r._bounds[1])).encode('ascii')
